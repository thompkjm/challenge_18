# Blockchain Validator

This application builds and validates simple transactions by storing them in a blockchain. It validates transactions by checking all of the hash links are correct.

---

## Technologies

Using Python 3.7

* [pandas](https://github.com/google/pandas) - For data analysis

* [streamlit](https://github.com/streamlit) - For web interface

---

## Installation Guide

```python
  pip install pandas
  pip install streamlit
```
